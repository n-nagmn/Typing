# 焼肉打タイピング (Yakiniku Type Master)

「焼肉」をテーマにした本格的なタイピングゲームです。流れてくるお肉の部位や焼肉に関連する単語を制限時間内にタイピングして、スコアを競い合います。

## 主な機能

- **選べる3つの難易度（コース）**: お手軽(3,000円)・おすすめ(5,000円)・高級(10,000円)から、実力に合わせたコースを選択可能。
- **多彩なプレイモード**:
  - **普通**: 標準的なスコアアタック。
  - **練習**: 制限時間なしで単語をタイピング。
  - **正確重視**: ミスタイプによるペナルティが大きいモード。
  - **速度必須**: 入力速度が遅いとコンボが途切れるモード。
  - **一発勝負**: 1度でもミスすると即終了の超ハードモード。
- **オンライン対戦**: リアルタイムで他のプレイヤーと同じお題をタイピングして競い合うマルチプレイモード。
- **ランキング機能**: コースやモードごとに保存されるスコアランキング（総合・週間・マイスコア対応）。
- **本格的なコンボシステム**: 連続入力成功でスコア倍率アップ（1.5倍、2.0倍など）。
- **タイピング履歴の可視化**: 「皿」の種類ごとの獲得枚数表示機能。

## 使用技術

- **フロントエンド**: HTML5 / CSS3 (Vanilla) / JavaScript (ES6+)
- **バックエンド**: Node.js, Express, Socket.IO
- **インフラ**: PM2 (デーモン化) / Nginx or Apache (静的ファイル配信)

## ローカルでの開発と実行

1. 本リポジトリをクローンします。
2. サーバーサイドの依存関係をインストールします。
   ```bash
   cd server
   npm install
   ```
3. バックエンドサーバーを起動します。
   ```bash
   node server.js
   ```
4. フロントエンド（`index.html`など）をブラウザで開くか、ローカルのWebサーバー（Live ServerやApache等）でホストしてアクセスします。

## サーバーへのデプロイ方法

本番環境（Ubuntu等）でのデプロイ例です。バックエンドはPM2を使用して常駐化します。

### 1. ソースコードの配置
`/var/www/html/Yakiniku` などの公開ディレクトリにリポジトリを配置します。

### 2. Node.js パッケージのインストール
```bash
cd /var/www/html/Yakiniku/server
npm install
```

### 3. PM2を使ったバックエンドサーバーの起動
Node.jsサーバー（ポート3019番）をバックグラウンドで起動・デーモン化します。

```bash
# PM2でサーバープロセスを起動
pm2 start server.js --name "yakiniku-server"

# プロセスの状態を確認
pm2 status

# （必要に応じて）PM2の自動起動を保存
pm2 save
```

**再起動や停止を行う場合:**
```bash
# 変更を反映するための再起動
pm2 restart yakiniku-server
# もしくはプロセスID(例: 20)を指定
pm2 restart 20

# ログの確認
pm2 logs yakiniku-server
```

### 4. クライアントからの接続先設定
フロントエンドからバックエンドへの接続先（APIおよびSocket.IO）は、`js/app.js` などの初期化部分で定義されています。
サーバーのIPアドレスまたはドメインに合わせて `http://<YOUR_SERVER_IP>:3019` のように設定してください。

## 遊び方

1. 画面中央の「コースとモード」を選択してスタートします。
2. 画面右から流れてくる単語のローマ字（ヘボン式・訓令式等に柔軟対応）を入力してください。
3. ミスなく連続で入力するとコンボが繋がり、獲得スコアがアップします（「ん」は `nn` や、末尾なら `n` 1回でもOKです）。
4. オンライン対戦では「オンラインロビー」からルームを作成、または既存のルームに参加することで複数人で対戦可能です。
