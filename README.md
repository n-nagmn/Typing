# 焼肉打タイピング (Yakiniku Type Master)

「焼肉」をテーマにした本格的なタイピングゲームです。流れてくるお肉の部位や焼肉に関連する単語を制限時間内にタイピングして、スコアを競い合います。

## 主な機能
- **シングルプレイ**: お手軽・おすすめ・高級の3コースから難易度を選択してプレイ
- **オンライン対戦**: リアルタイムで他のプレイヤーとスコアやタイピングスピードを競える対戦モード
- **ランキング機能**: 世界中のプレイヤーとスコアを競うランキング機能
- **結果・コンボシステム**: 連続成功でスコア倍率アップなど、本家にインスパイアされた本格的なスコア計算

## 使用技術
- HTML / CSS / Vanilla JavaScript
- Node.js (Express, Socket.IO) - オンライン対戦・ランキング用バックエンド
- Nginx / PM2

## 遊び方
1. コースを選択してスタートします。
2. 画面右から流れてくる単語のローマ字を入力してください。
3. ミスなく連続で入力するとコンボが繋がり、獲得スコアがアップします。
4. オンライン対戦では、ルームを作成または参加することで複数人で対戦可能です。
